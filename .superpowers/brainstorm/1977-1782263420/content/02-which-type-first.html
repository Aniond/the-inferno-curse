Single Target (Attack / Heal)

Direct single-cell effects. Easiest to implement first using existing attack/targeting code. Good for Normal abilities.

Focus areas: CT steal/boost, simple damage/heal, basic status
2

Area Effect Spells

Shapes on the grid (cross, 3x3, line, cone). Needs new targeting logic but high tactical payoff. Strong Uncommon candidate.

Focus areas: AoE shapes, height interaction, weather dampening, multi-target CT
3

Buff / Curse

Status-like effects on CT, power, movement. Leverages existing apply_status_effect and modifiers. Fits both Normal and Uncommon.

Focus areas: Duration, stacking, CT multiplier/flat changes, corruption on use
4

Traits (Passives)

Always-on or equipped effects. Rare tier mostly. Impacts builds and class identity. Needs careful limits.

Focus areas: Passive hooks in combat formulas, equip slots, theme tags for jobs
