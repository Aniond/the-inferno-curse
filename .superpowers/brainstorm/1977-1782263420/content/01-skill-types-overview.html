The Inferno Curse: 4 Core Skill Types

Designing abilities for the monster absorption system • Tactical grid (CT/action turns, height, facing, weather) • Normal / Uncommon / Rare tiers

Context from current systems
  • Abilities are string IDs today (e.g. "Strike", "Guts") — will expand to typed data
  • Combat is action-driven CT: every move/rotate/act advances everyone's CT. Skills can directly +/- CT, apply status, change multipliers
  • Grid is cell-based with height, cover, facing arcs, LOS
  • Weather and status already provide combat mods (ct_gain_mult, move, damage, LOS)
  • Absorption: Normal (frequent, simple), Uncommon (tactical shift), Rare (build-defining passive or powerful)
SINGLE TARGET
Attack or Heal • 1 cell

1. Single Target (Attack / Heal)

Direct damage, single ally heal, simple elemental poke, basic status (poison 1 target), low-cost utility (scan, minor CT boost).

• Typical Normal tier
• Simple targeting (existing attack code)
• Can directly modify target.current_ct or self CT
• Example: "Strike" (+dmg, -15 CT on target) or "Guarigione" (+HP)
AREA EFFECT
Cross / 3x3 / Cone • Multiple cells

2. Area Effect Spells

Area damage (3x3, cross, line, cone), area heal, zone control (create difficult terrain), multi-target status, forced movement (push/pull).

• Typical Uncommon tier
• Needs grid query for affected cells
• Height can modify (e.g. blast from high ground spreads less)
• Weather can reduce (rain dampens fire AoE)
• Example: "Ash Hex" (3x3 corruption zone + damage over time)
BUFF
CURSE
CT +20% CT -30%

3. Buff / Curse

Self or ally buff (speed, defense, CT gain), enemy debuff (slow CT, reduced power, status like paralyze/fear), morale effects, temporary height or cover bonuses.

• Can be Normal (weak) or Uncommon (strong tactical)
• Apply via existing status_effect + ct_modifiers
• Duration in rounds or actions
• Example: "Dirty Trick" (curse enemy CT gain -40% for 2 rounds)
PASSIVE / TRAIT
Counterattack
When struck in melee, 30% chance to auto-strike back for 50% damage.
Always on • No CT cost • Rare only

4. Traits (Passives)

Passive stat mods, resistances, counter rules, healing boost, movement rules, formation bonuses, conditional triggers (bonus vs isolated), morale/passive defense while holding position.

• Almost exclusively Rare tier
• Equipped / always active (limited slots?)
• Can modify core formulas (get_power, get_step_cost, etc.)
• Shape class identity and corruption
• Example: "Limbo Echo" (preserve from death once per battle)
