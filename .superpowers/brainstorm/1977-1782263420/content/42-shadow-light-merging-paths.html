Orb Tree Update: One Equipped per Type + Shadow/Light Merging Paths

Confirmed: Only ONE copy of each unique orb can be equipped at a time, even with multiples in the tree. Tree now has branching Shadow (Sinner) and Light (Saint) paths that can merge for hybrid orbs.

Equip Limit Rule (your reminder)

For any unique orb (e.g. "Shadow Claw"):

  • You can place unlimited copies in the tree to stack power and sanity cost.
  • You can only ever have ONE instance equipped for active use.
  • The power from ALL stacked copies applies to that single equipped version.

This keeps loadouts meaningful while still letting you farm and invest heavily in favorites.

Shadow / Light Branching with Merge Points

Tree Flow (simplified)

Base Orb Entry
↓
Split:
  Shadow Branch (Sinner) — raw power, higher base sanity + corruption, aggressive effects
    ↓
    Merge Nodes (hybrid options appear here)
    ↑
  Light Branch (Saint) — purified lean, lower corruption, utility or holy effects, Church synergy

Merge Examples
• Merge a Shadow Damage orb with Light nodes → gains holy damage vs Limbo creatures + small self-bless on hit, but base damage slightly reduced and sanity cost stays high.
• Merge a Light Heal orb with Shadow nodes → "Blood Sacrament" style: heals allies but drains a small amount from enemies or self, adds corruption bleed on use.
• Deep merges can create unique hybrid orbs that are neither pure dark nor pure light.

This replaces the strict "Dark and Light cannot coexist" with player choice at merge points.

Impact on Single Target Template

Each Single Target orb (Damage or Heal) now tracks:

  • Its flat base_sanity_cost and base_corruption_value.
  • Path investment: pure Shadow, pure Light, or merged hybrid (chosen via tree nodes).
  • Only one version can be equipped, but it receives power from all tree stacks on that orb (including any merged nodes).

Example flow for Shadow Claw:

  • Start in Shadow branch → high damage, high sanity/corruption.
  • Reach a merge node → option to add Light elements (holy vs Limbo, reduced world corruption on use, but lower raw damage and retained high sanity cost).
  • Equip the resulting hybrid — only one active, powered by everything you've slotted.

Benefits

  • Enforces the "only one equipped per orb type" while still rewarding heavy tree investment via stacking.
  • Gives real choice and replayability through merge points without losing the Saint/Sinner identity.
  • Hybrids create interesting orbs that feel unique to the player's path decisions.
  • Church can still "reset" an orb toward pure Light if the player wants to back out of deep Shadow investment.
  • Fits the "equip one, invest many" rule you confirmed.

Tradeoffs to balance

  • Merge points need clear costs/benefits so pure paths remain attractive.
  • Need to decide if hybrids are "best of both" or always have a downside (e.g. higher sanity cost than pure Light but lower than pure Shadow).
